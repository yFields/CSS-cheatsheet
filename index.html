Common CSS Attributes

Attribute Values Description
color e.g black,rgba(),rgb() Sets the foreground color of the selected Object to the chosen color.
background-color e.g black,rgba(),rgb() Sets the background color of the selected Object to the chosen color.
opacity 0-1 Determines how transparent an Object should be (0 fully transparent and 1 not transparent at all
font-family eg. Helvetica, Georgia, Aerial etc. Determines which font should be used for Text within the selcted element
font-size eg. 10px, 20px, 3em Determines the font size used for text in the selceted element
font-weight eg. bold, bolder Determines the weight of the font within an element. Is used to make text look bold for example.
padding 10px; 10px 20px; 10px 15px 25px 12px Determines how much space should be between the border of the selected element and the actual data within the element
margin 10px; 10px 20px; 10px 15px 25px 12px Determins how much space there should be between the selceted element and other elements surrounding it.
background-image url('./link/to/your/image') Selects an image to be used as the background of the selected element.

CSS Selectors

Syntax Values Description
selector {} eg. h1, p, body This selector selects all HTML tags in a file to apply the specified styling to. It has the lowest specifity of all selctors.
.selector {} any class name This selecotr selects all HTML elements with the specified class attribute. It is more specific than the tag selector but less specific than the ID selector
#selector {} any ID name This selector selects the single HTML element with the specified id attribute. It has the highest specifity of all selectors.
